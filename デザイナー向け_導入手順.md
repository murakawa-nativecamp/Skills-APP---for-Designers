# デザイナー向け：スキルの入れ方（ていねい版）

このスキル集（Figmaデザイン修正など）を Cowork で使えるようにする手順です。
GitHub の知識はいりません。**画面のボタンを押すだけ**です。

貼り付けに使う文字列はこれ1つだけ覚えればOKです：

```
murakawa-nativecamp/Skills-APP---for-Designers
```

---

## ⭐ おすすめ：ルートA（管理者が1回設定 → デザイナーは押すだけ）

チームで使うなら、これが一番ラクで確実です。
**会社の管理者（Owner）が最初に1回だけ**設定すれば、デザイナー全員の画面に自動で出てきます。

### 管理者がやること（1回だけ）

1. ブラウザで Claude にログインし、**組織の設定（Settings）** を開く
2. **Plugins（プラグイン）** のページを開く
3. **「Add plugins」**（プラグインを追加）→ **「GitHub」** を選ぶ
4. リポジトリ欄に次を貼り付ける：
   `murakawa-nativecamp/Skills-APP---for-Designers`
5. 追加したら、そのマーケットプレイスの右上メニューで
   **「Sync automatically」（自動同期）** をオンにする
   → これで、あなたが更新するたびにデザイナー側も自動で最新になります

### デザイナーがやること

1. Cowork 左の **「Customize」** を押す
2. **「Plugins」** タブを開く
3. **「Your organization」**（あなたの組織）タブに
   **nativecamp-design-skills** が出ているので、**「Install」** を押す
4. 完了！

---

## ルートB：デザイナーが各自で追加する（管理者設定なしの場合）

管理者設定をしない場合は、デザイナー一人ひとりが次をやります。

### ① 最初に1回だけ：マーケットプレイスを追加

1. Cowork の左サイドバーにある **「Customize」** ボタンを押す
2. 開いたパネルの中の **「Plugins」** タブを押す
3. **「Personal」**（個人）タブを選ぶ
4. **「＋」** ボタンを押す
5. メニューから **「Add marketplace」**（マーケットプレイスを追加）を選ぶ
6. 入力欄に、次の文字列をそのまま貼り付ける：
   ```
   murakawa-nativecamp/Skills-APP---for-Designers
   ```
7. **「Add」** を押す
8. **「Done」** を押して閉じる

### ② プラグインをインストール

1. 追加されたマーケットプレイスの中に
   **nativecamp-design-skills** が表示される
2. その **「Install」** ボタンを押す
3. 完了！

> うまく「Add marketplace」が見つからない・動かない場合は、ルートA（管理者設定）をお試しください。そちらが確実です。

---

## 使い方（インストール後）

特別な操作はいりません。いつもどおり会話で頼むだけです。

- 例：「この参考画面に合わせて作成画面をDSで直して」と頼む
  → 関連するスキルが自動で働きます
- チャット入力欄で **「/」（スラッシュ）** を打つと、使えるスキルの一覧が出ます

---

## 更新（最新版を取り込む）

- **ルートA（管理者が自動同期オン）の場合**：何もしなくてOK。自動で最新になります。
- **ルートB（個人で追加）の場合**：チャットで次のコマンドを送ると最新になります：
  ```
  /plugin marketplace update murakawa-nativecamp/Skills-APP---for-Designers
  ```

---

## 困ったときは

- 「Customize」や「Plugins」が見当たらない → Cowork のバージョンが古い可能性。アプリを最新に更新してください。
- それでも分からない → 村川までご連絡ください。
